Overview

Tower Defence Game is a strategy-based game developed using Unity.
The objective of the game is to defend a base by placing defensive towers that automatically attack incoming enemies. Players must strategically position towers and manage resources to stop waves of enemies from reaching the target.

Tower defense games typically require players to plan tower placement and upgrades carefully in order to handle progressively stronger enemy waves.

This project demonstrates gameplay mechanics such as enemy pathing, tower targeting systems, and wave-based enemy spawning.

Features

Wave-based enemy spawning system

Tower placement and attack mechanics

Enemy path-following system

Targeting logic for attacking enemies

Strategy-based gameplay mechanics

Gameplay

The player must build towers along the enemy path to prevent enemies from reaching the base.

Basic gameplay loop:

Enemies spawn in waves.

The player places towers on designated positions.

Towers automatically attack enemies within range.

Each wave increases difficulty.

The game continues until enemies breach the defense.
